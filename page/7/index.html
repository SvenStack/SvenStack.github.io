A valid BST is defined as:

  • The left subtree of a node contains only nodes with values less than the node‚Äôs value.
  • The right subtree of a node contains only nodes with values greater than the node‚Äôs value.
  • Both the left and right subtrees must also be valid BSTs.

üí¨ Example

üí° Approach 1: DFS with Value Ranges

We recursively check each node‚Äôs value against an allowed range:

  1. Initially, the root has range (-‚àû, +‚àû).
  2. For left child: upper bound is parent‚Äôs value.
  3. For right child: lower bound is parent‚Äôs value.
  4. If any node violates the range rule, it‚Äôs not a BST.

üí° Approach 2: Inorder Traversal Check

  • Perform inorder traversal of the tree.
  • If the resulting sequence is strictly increasing, it‚Äôs a valid BST.

üî¢ Java Code

‚è± Time and Space Complexity

  • Time Complexity: O(n) ‚Äî every node visited once.
  • Space Complexity: O(h) ‚Äî recursion depth (h is tree height).

‚úçÔ∏è Summary

  • Range-check DFS is the most intuitive and avoids extra storage.
  • Inorder traversal works because BST‚Äôs inorder is sorted.
